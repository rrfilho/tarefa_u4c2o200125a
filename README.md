# Tarefa 3 Unidade 4 (20/01)
Microcontroladores - GPIO

Para cobrir os requisitos estabelecidos nas instruções da **[Tarefa 1 U4C2 (20/01)](https://moodle.embarcatech.cepedi.org.br/pluginfile.php/2320/mod_assign/introattachment/0/TAREFA%20-%20%20AS_U4C2O1234A_20_01_2025%20-%20V1.pdf?forcedownload=1)** implementou-se um sistema em que as teclas de 1 a 9 do teclado numérico (keypad) inicializam uma animação diferente com mais de 5 frames. Além disso, as teclas de 'A', 'B', 'C', 'D' e '#' foram programadas para também acionar a matriz de LEDS da maneira especificada no documento.
